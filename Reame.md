# Mediator API

O Mediator API é um microserviço que atua diretamente com a aprovação de ideias de projeto(curadoria), antes de serem publicadas na plataforma e disponibilizadas para o mediador. Ele é responsável por receber as ideias de projeto, realizar a curadoria e aprovar ou rejeitar as ideias, garantindo que apenas as ideias de qualidade sejam publicadas na plataforma.
